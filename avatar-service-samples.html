Live samples below load straight from each service's public CDN/API (no key, no build). LIVE = rendered here · BROWSE = open the gallery (not hot-linkable). If a tile is blank, that asset failed to load — note it, the others still tell you the style.

① Status-indicator demo LIVE · Lottie

This is exactly your use case — animated Lottie mapped to a detected state. Source: Google Noto Animated Emoji (open source, Apache-2.0). Each is a real .lottie/.json on a stable Google CDN.
Wire-up: lottie.loadAnimation({ path: `…/${codepoint}/lottie.json` }) — swap the codepoint per detected status.

② DiceBear LIVE · API

FreeNo keySVG 30+ stylesDeterministic by seed
Easiest wire-up of all — it's just an <img>. Each style + seed is stable & cacheable. Covers Open-Peeps, Avataaars, Notion-style and more. dicebear.com
https://api.dicebear.com/9.x/{style}/svg?seed={seed}
Same seed = same face forever (great for "this person"). Add &backgroundColor=b6e3f4, &radius=50 etc. to brand them.

⑧ Illustration & animated-icon libraries BROWSE

These need a quick download / customize step (no clean hot-link API), but they're the highest-quality "professional cartoon" sources. Open, pick, self-host on your CDN.
LottieFilesHuge free Lottie library — coding, coffee, typing, loaders. Per-asset Free/Pro license. LordiconAnimated icons w/ hover/loop states — built for UI status. ~500 free w/ attribution. UseAnimations~100 free animated icons (Lottie/JSON). Loading, activity, toggles. unDrawOpen SVG illustrations, recolor to brand, NO attribution. The safe default.
